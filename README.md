This project is a simple landing page created as part of **The Odin Project** curriculum.

The goal of this project was to practice building a complete webpage using HTML and CSS while applying modern layout techniques such as Flexbox. During this project, I focused on writing clean, organized code and improving my understanding of webpage structure and styling.
